# ✈️ Flight Price Prediction using Data Science

## 📌 Project Overview

This project focuses on predicting flight ticket prices using end-to-end data science techniques. The dataset includes features like airline, journey details, duration, and price.

The goal is to analyze the data and build machine learning models to accurately predict flight prices.

---

## 🚀 Technologies Used

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* Plotly
* Scikit-learn
* Google Colab

---

## 🔍 Project Workflow

### 1. Data Collection and Import

* Loaded dataset using Pandas
* Explored dataset structure

### 2. Data Cleaning

* Removed duplicates
* Handled missing values
* Fixed data formats

### 3. Feature Engineering

* Used processed dataset
* Separated features and target
* Applied scaling

### 4. Exploratory Data Analysis (EDA)

* Generated statistical summaries
* Analyzed relationships and distributions

### 5. Data Visualization

* Created histograms, boxplots, scatter plots
* Identified trends and outliers

### 6. Interactive Visualization

* Built interactive charts using Plotly

### 7. Regression Model

* Applied Linear Regression
* Evaluated using MAE and R² score

### 8. Classification Models

* Converted price into categories
* Used Decision Tree and Random Forest

### 9. Clustering

* Applied K-Means clustering
* Grouped flights into clusters

### 10. Capstone Project

* Combined all steps into a complete workflow

---

## 📊 Results

* Successfully predicted flight prices
* Achieved good model performance using regression
* Classified prices using machine learning
* Identified patterns using clustering

---

## 🎯 Conclusion

This project demonstrates a complete data science pipeline including data preprocessing, visualization, and machine learning. It shows that flight prices can be effectively predicted using data-driven approaches.
